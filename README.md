# E-Commerce Sales Analytics using SQL

## Project Overview

This project analyzes the Olist Brazilian E-Commerce Dataset using PostgreSQL.

The objective is to generate business insights related to customer behavior, order trends, payment methods, product performance, seller performance, and customer satisfaction.

---

- PostgreSQL
- pgAdmin 4
- SQL
- GitHub

## Dataset Information

Dataset: Olist Brazilian E-Commerce Dataset

Tables Used:

1. Customers
2. Orders

4. Sellers

6. Payments
7. Reviews
8. Categories
